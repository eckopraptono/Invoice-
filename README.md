# Invoice-

A simple, customizable invoice template for DotCom Partner, built with HTML, CSS, and JavaScript. This template allows you to create professional invoices with editable fields, export to PDF, and manage payment statuses.

## Features

- Editable invoice details (number, date, due date)
- Client information input
- Itemized billing with automatic calculations
- Payment status tracking (Waiting, Paid, Cancelled)
- PDF export functionality
- Responsive design
- Indonesian language support

## Usage

1. Open `invoice-dotcom.html` in a web browser
2. Fill in the invoice details, client information, and items
3. Update the payment status as needed
4. Export to PDF using the "Export PDF" button

## Technologies Used

- HTML5
- CSS3 (with custom properties)
- JavaScript (for interactivity and PDF generation)
- html2pdf.js library for PDF export

## Customization

You can customize the template by editing the HTML, CSS, and JavaScript in `invoice-dotcom.html`. The design uses CSS custom properties for easy theming.
